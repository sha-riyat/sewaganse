# Sewa Ganse — Portfolio

Bienvenue sur le dépôt de mon portfolio personnel. Ce projet est une vitrine de mes compétences en tant que développeur web et mobile, mettant en avant mes projets, mon expertise et ma vision du développement numérique.

## 🚀 La Stack Technique

Ce portfolio est bâti sur une stack moderne, axée sur la performance et l'expérience utilisateur (UX) :

- **[Astro](https://astro.build/)** : Framework web "static-first" pour une vitesse de chargement optimale et un SEO performant.
- **TypeScript** : Pour un code robuste, typé et maintenable.
- **Vanilla CSS** : Design épuré et performant sans dépendances de frameworks CSS lourds.
- **Architecture de Composants** : Utilisation des composants Astro pour une modularité maximale.

## ✨ Fonctionnalités

- **Performance 100/100** : Optimisation poussée pour des scores Lighthouse parfaits.
- **Bilingue (FR/EN)** : Gestion du changement de langue avec persistance via `localStorage`.
- **Design Responsive** : Adapté à tous les écrans, du mobile au desktop ultra-large.
- **Gestion Dynamique des Projets** : Les projets sont centralisés dans un fichier de données TypeScript pour une mise à jour facile.

## 🛠️ Installation et Développement

### Prérequis

- **Node.js** (v18.0.0 ou supérieure)
- **npm** ou **yarn**

### Étapes

1. **Cloner le projet**
   ```bash
   git clone https://github.com/votre-username/sewa-ganse-dev.git
   cd sewa-ganse-dev
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```
   Votre site sera accessible sur `http://localhost:4321`.

4. **Builder pour la production**
   ```bash
   npm run build
   ```

## 📂 Structure du Projet

```text
/
├── src/
│   ├── assets/       # Images et ressources statiques
│   ├── components/   # Composants réutilisables (Header, Footer, Cards)
│   ├── data/         # Contenu des projets (projects.ts)
│   ├── pages/        # Routes et pages du site
│   ├── scripts/      # Logique client-side (gestion des langues)
│   └── styles/       # Fichiers CSS globaux et variables
├── public/           # Fichiers statiques publics
└── astro.config.mjs  # Configuration d'Astro
```
